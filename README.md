# 🎬 哔哩哔哩（bilibili.com）调整

> 一个简单的用户脚本，主要是给自己用的。

当前版本为 V3，已完全重构并新增多项功能。因脚本托管网站（如 GreasyFork）不允许上传混淆压缩后的代码，该版本将不再通过脚本托管网站发布。  

---

## ✨ 功能概览

- **浏览器全适配** — 已适配 Chrome、Firefox 等主流浏览器
- **独立设置面板** — 无需依赖脚本管理器菜单，页面内直接修改设置
- **自动更新提醒** — 托管在个人服务器，支持自动检查更新并弹窗提醒
- **AI 广告跳过** — 使用 AI 识别视频中的广告片段并自动跳过（需视频有 AI 字幕）
- **评论区增强** — IP 属地展示、标签移除、视频简介嵌入
- **播放体验优化** — 自动定位播放器、画质选择、网页全屏解锁、自动暂停/恢复
- **首页辅助** — 推荐视频历史记录、付费视频标记

> **注意**：利用空闲时间开发维护，虽已长期使用未发现重大问题，但仍无法保证完全没有 Bug。若使用中遇到问题请反馈，会尽快处理。

## 🛠 功能列表

### 首页
- **推荐视频历史记录** — 记录前 11 个非广告推荐位，防止误点刷新错过想看的视频，支持搜索、按批次排序
- **付费视频识别** — 调用播放器 API 检测付费视频，在卡片标题中实时添加「付费视频」标记

### 动态页
- **默认显示「投稿视频」** — 进入动态页自动切换至投稿视频 tab

### 播放页
1. **自动定位播放器** — 进入播放页自动滚动至播放器，支持偏移量设置
2. **播放器模式设置** — 可自定义默认模式（宽屏/网页全屏/关闭）
3. **画质选择** — 支持自动选择最高画质（含 4K/8K 选项）
4. **快速返回按钮** — 悬浮按钮，一键返回播放器
5. **时间锚点** — 点击评论区时间锚点快速跳转至播放器
6. **网页全屏解锁** — 全屏模式下可滚动查看评论，新增「跳转至评论区」按钮
7. **视频简介优化** — 替换静态文本为可跳转链接，并嵌入评论区
8. **自动暂停/恢复** — 离开页面时暂停视频，返回时自动恢复播放
9. **自动字幕** — 自动开启「中文 AI」字幕
10. **IP 属地展示** — 评论区显示用户 IP 归属地
11. **标签移除** — 移除评论中「UP 主觉得很赞」等标签
12. **AI 广告跳过** — 使用 AI 识别广告片段并自动跳过（需视频有 AI 字幕）
